# Лабораторная работа #1

Составить список требований, предъявляемых к разрабатываемому веб-сайту:

> Дельфийский оракул — всевозможные онлайн гадания - [https://magya-online.ru/](https://magya-online.ru/)

Требования должны делиться на следующие категории:

- **Функциональные**
  - Требования пользователей сайта
  - Требования владельцев сайта
- **Нефункциональные**

Требования необходимо оформить в соответствии с шаблонами RUP _(документ SRS - Software Requirements Specification)_. Для каждого из требований нужно указать его атрибуты _(в соответствии с методологией RUP)_, а также оценить и аргументировать приблизительное количество часов, требующихся на реализацию этого требования.

Для функциональных требований нужно составить UML UseCase-диаграммы, описывающие реализующие их прецеденты использования.
