\subsubsection{User Characteristics}

Данный раздел описывает общие характеристики пользователей, которые будут использовать программный продукт.

Система предусматривает две категории пользователей:

\begin{enumerate}
    \item \textbf{Пользователь (посетитель Сайта, гость)}
    \begin{itemize}
        \item \textbf{Образование и опыт:} любой уровень. Обладает базовыми навыками работы в сети Интернет и веб-браузере.
        \item \textbf{Техническая экспертиза:} отсутствует. Пользователь не знает о внутреннем устройстве программного продукта.
        \item \textbf{Влияние на требования:} данная характеристика обуславливает необходимость создания интуитивно понятного интерфейса, а также четких и понятных инструкций по использованию гаданий.
    \end{itemize}
    \item \textbf{Администратор}
    \begin{itemize}
        \item \textbf{Образование и опыт:} любой уровень. Уверенный пользователь ПК и сети Интернет.
        \item \textbf{Техническая экспертиза:} минимальная. Администратор не владеет навыками программирования и не имеет доступа к серверной части программного обеспечения.
        \item \textbf{Влияние на требования:} данная характеристика обуславливает необходимость создания удобной панели управления, позволяющей управлять содержимым Сайта без вмешательства разработчика.
    \end{itemize}
\end{enumerate}
