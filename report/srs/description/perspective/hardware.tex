\subsubsubsubsection{Hardware Interfaces}

Сайт функционирует на стандартном серверном оборудовании, предоставляемом используемым хостинг-провайдером. Характеристики аппаратных интерфейсов задаются следующим:

\begin{itemize}
    \item \textbf{Серверное оборудование:} архитектура x86-64, сетевой интерфейс с выделенным IP-адресом и поддержка протоколов TCP/IP.
    \item \textbf{Клиентское оборудование:} устройство пользователя с выходом в сеть Интернет. Взаимодействие с Сайтом происходит через стандартные драйверы устройств.
\end{itemize}
