\subsubsubsubsection{User Interfaces}

Пользовательский интерфейс Сайта представляет собой совокупность веб-страниц, выстроенных по принципу иерархической навигации от общего к частному. Все веб-страницы единообразны по визуальным решениям и адаптивны к различным типам устройств.

Предусмотрены два типа пользовательских интерфейсов, физически отделеных друг от друга:

\begin{itemize}
    \item \textbf{Публичный интерфейс} --- предназначен для всех посетителей Сайта.
    \item \textbf{Административный интерфейс (панель управления)} --- предназначен для ограниченного круга лиц, имеющих доступ к редактированию контента Сайта.
\end{itemize}
