\subsubsubsubsection{User Interfaces}

Пользовательский интерфейс Сайта --- совокупность веб-страниц, выстроенных по принципу иерархической навигации от общего к частному. Они имеют модульную структуру и должны включать следующие сквозные элементы:

\begin{itemize}
  \item Шапка (Header)
  \item Основная область (Content Area)  
  \item Подвал (Footer)
\end{itemize}

Компоновка основной области зависит от типа представляемого контента. Рекомендуется использовать следующие варианты:

\begin{enumerate}
  \item \textit{Одноколоночный макет:} применяется исключительно для страниц публикаций. Это помогает убрать визуальный шум при чтении текстовых материалов и акцентировать внимение на контенте.
  \item \textit{Двухколоночный макет:} применяется во всех иных случаях. Правая колонка шире левой, она содержит основной контент. Левая колонка обычно содержит дополнительную информацию для читателя.
\end{enumerate}

Вёрстка страниц Сайта должна изменяться так, чтобы возможности сайта и контент были одинаково доступны на персональных компьютерах и мобильных устройствах. Для этого следует учитывать ширину экрана используемого Пользователем устройства при написании стилей.
