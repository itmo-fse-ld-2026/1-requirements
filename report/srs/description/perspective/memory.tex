\subsubsubsubsection{Memory Constraints}

Для функционирования Сайта должны использоваться стандартные объемы оперативной и постоянной памяти, предоставляемые в базовым пакете используемого виртуального хостинга. При росте популярности Сайта и, соответственно, нагрузки требования к памяти могут быть пересмотрены и согласованы с заказчиком.
