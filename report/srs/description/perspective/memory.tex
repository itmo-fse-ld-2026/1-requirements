\subsubsubsubsection{Memory Constraints}

Для определения минимального объёма постоянной и оперативной памяти, необходимой для функционирования Сайта, требуется учитывать следующие факторы:

\begin{itemize}
  \item Сайт не хранит состояние Пользователей, потому что на нём отсутствует возможность регистрации и активные сессии.
  \item Суточная аудитория сайта ожидается в районе 30 000 уникальных Пользователей, каждый из них посещает в среднем по 7 страниц.
  \item Сайт предоставляет преимущественно статический контент, который также требуется хранить.
  \item Сайт не выполняет фоновых задач, что исключает дополнительное потребление памяти в отсутствие пользовательской активности.
  \item Провайдер хостинга может не иметь возможности предоставить необходимый объём ресурсов.
\end{itemize}

Ограничения выше могут меняться с течением времени, поэтому требования к памяти могут быть пересмотрены и согласованы с заказчиком.
