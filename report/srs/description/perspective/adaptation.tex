\subsubsubsubsection{Site adaptation requirements}

Сайт разрабатывается для русскоязычной аудитории. Весь контент, включая элементы интерфейса, должен быть реализован на русском языке.

При первом запуска веб-приложение должно быть инициализировано стартовым набором данных, предоставленным заказчиком.
