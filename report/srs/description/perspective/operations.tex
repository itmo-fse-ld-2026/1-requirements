\subsubsubsubsection{Operations}

Для Сайта определены два режима работы:

\begin{itemize}
    \item \textbf{Интерактивный режим (основной):} Сайт обрабатывает запросы пользователей в режиме реального времени без остановок, за исключением проведения технических работ. Пользователи имеют доступ к полной функциональности Сайта.
    \item \textbf{Режим обслуживания:} Администратор может обновлять контент Сайта с помощью панели управления, не блокируя при этом доступ пользователей ко всей функциональности Сайта.
\end{itemize}
