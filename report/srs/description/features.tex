\subsubsection{Product Functions}

Данный раздел кратко описывает основные функции, которые будет выполнять программный продукт.

Сайт «Дельфийский оракул» обеспечивает пользователям возможность просматривать категоризированную коллекцию интерактивных гаданий с подробными описаниями и инструкциями, получать предсказания, оценивать и комментировать их, видеть комментарии других пользователей, делиться конкретными гаданиямм в социальных сетях. Также реализована защищенная панель управления, предоставляющая доступ к редактированию содержимого Сайта и модерированию комментариев.
