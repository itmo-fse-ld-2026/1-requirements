\subsubsection{Product Functions}

Данный раздел кратко описывает основные функции, которые будет выполнять программный продукт.

Сайт «Дельфийский оракул» обеспечивает следующую функциональность:

\begin{itemize}
    \item Просмотр категоризированной коллекции интерактивных гаданий с подробными описаниями и инструкциями.
    \item Получение предсказаний (результатов гаданий).
    \item Оценивание гаданий.
    \item Просмотр и добавление комментариев.
    \item Возможность поделиться конкретным гаданием в социальных сетях (ВКонтакте, Telegram, Одноклассники).
    \item Доступ к защищенной панели управления для редактирования содержимого сайта и модерации пользовательских комментариев.
\end{itemize}
