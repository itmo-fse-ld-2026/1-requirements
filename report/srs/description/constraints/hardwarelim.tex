\subsubsubsubsection{Hardware Limitations}

Данный программный продукт наделен следующими аппаратными ограничениями:
\begin{itemize}
  \item Суточная аудитория в 30 000 уникальных Пользователей обязывает учитывать пропускную способность сетевого канала, которую готов предоставить провайдер хостинга для обработки такого потока запросов.
  \item Используемое программное обеспечение требует 64-битной архитектуры процессора, на котором будет работать хостинг.
\end{itemize}
