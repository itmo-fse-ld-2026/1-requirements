\subsubsubsubsection{Hardware Limitations}

Данный программный продукт наделен следующими аппаратными ограничениями:
\begin{itemize}
    \item Сайт должен функционировать в рамках ресурсов, предоставляемых выбранным хостингом.
    \item На начальном этапе предполагается использование виртуального хостинга с ограниченными системными ресурсами.
\end{itemize}
