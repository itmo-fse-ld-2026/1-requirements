\subsubsubsubsection{Reliability Requirements}

Заказчик ожидает уровень доступности Сайта не ниже 99.5\% времени в месяц. Плановые технические работы, требующие временной недоступности Сайта, должны быть предварительно согласованы с заказчиком и проводиться в периоды наименьшей предполагаемой активности пользователей.
