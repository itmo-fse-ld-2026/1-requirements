\subsubsection{Purpose}

Данный документ содержит исчерпывающий перечень требований к сайту в сети Интернет «Дельфийский оракул». Здесь приведено детальное описание текущей архитектуры сайта и ограничений, которые накладываются на Интернет-ресурс в связи с особенностями публикуемой на нём информации и целевой аудитории.

Ожидается, что текущий документ окажется полезен для следующих групп стейкхолдеров:
\begin{itemize}
  \item \textit{пользователи} получат список всех возможностей, которые предоставляются текущим Интернет-ресурсом;
  \item \textit{сопровождающая сторона} получит описание системы, функционирование которой необходимо поддерживать.
\end{itemize}
